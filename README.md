# Telecom X - Previsão de Churn

Projeto desenvolvido no programa Oracle Next Education (ONE) em parceria com a Alura.

## Objetivo

Construir modelos de Machine Learning capazes de prever a evasão (churn) de clientes da empresa Telecom X, identificando fatores que influenciam o cancelamento de serviços.

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Etapas do projeto

1. Carregamento dos dados tratados
2. Análise exploratória (EDA)
3. Tratamento e codificação das variáveis
4. Separação treino/teste
5. Treinamento de modelos preditivos
6. Avaliação com métricas de classificação
7. Análise de importância das variáveis

## Modelos utilizados

- Regressão Logística
- Random Forest

## Principais insights

- Clientes com contratos mensais apresentam maior probabilidade de churn
- Tempo de permanência (tenure) tem forte relação com retenção
- Serviços adicionais como suporte técnico reduzem a evasão
